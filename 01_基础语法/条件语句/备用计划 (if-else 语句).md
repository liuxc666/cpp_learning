if-else 结构：它提供了一个二选一的路径。程序要么执行if块，要么执行else块，绝不会两个都执行，也绝不会两个都不执行。
- 1.bool：布尔数据类型，只能存储true或false。在进行逻辑判断时非常核心。
	
- 2.if 语句：if (condition) { code_to_execute; }。这是C++中最基本的决策结构。
	
- 3.==（等于运算符）：注意！这是两个等号！它用于**比较**左右两边的值是否相等，返回true或false。一个等号=是**赋值运算符**，用于把右边的值赋给左边的变量。这是初学者最常犯的错误之一！把if (a == 5)写成if (a = 5)会导致逻辑错误。
