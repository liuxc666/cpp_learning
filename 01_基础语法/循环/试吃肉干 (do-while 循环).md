【do-while 循环的标准格式 / Standard Format】
```
do {
    // Code to be executed at least once
    // 至少执行一次的代码
} while (condition); // Note the semicolon at the end!
                      // 注意结尾处的分号！
```
do-while 循环会先执行一次 do 后面的代码块，然后再去检查 while 括号里的条件。如果条件为 true，就返回去继续执行 do 代码块，如此往复。

- char: 这是一个关键字，代表“character”（字符）。它是一种数据类型，专门用来存储单个字符。比如字母'a'，数字'1'，符号'?'。它占用的内存空间非常小，通常只有一个字节。
