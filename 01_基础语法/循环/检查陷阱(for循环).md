```cpp
for (initialization; condition; update) {

  // Code to be executed in each iteration

  // 每次迭代中要执行的代码

}
```

- initialization (初始化): 在循环开始前**只执行一次**的语句。通常用来声明和初始化一个计数器变量，例如 int i = 0;。

- condition (条件): 在**每次循环开始前**都会检查这个条件。如果条件为 true，循环体内的代码就会执行；如果为 false，循环就会终止。

- update (更新): 在**每次循环结束后**执行的语句。通常用来增加或减少计数器的值，例如 i++ (这等同于 i = i + 1)。
